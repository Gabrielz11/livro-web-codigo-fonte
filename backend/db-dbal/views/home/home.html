Select em cascata

Uma das grandes dificuldades em aplicativos é criar os chamados "selects em cascata". O contexto de dados de cidades (que pertencem a Estados) demonstra bem este tipo de problema quando é necessário, por exemplo, com dois selects, selecionar as cidades de um Estado selecionado anteriormente.

O exemplo abaixo está consultando o banco de dados para realizar esta tarefa.
